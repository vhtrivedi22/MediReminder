MediReminder

Your automated medication reminder system

Project Description

MediReminder is a simple web-based tool that helps users remember to take their medications on time. By integrating Google Forms, Google Sheets, Google Calendar, and Gmail, it automatically schedules reminders and sends confirmation emails to users. This project showcases automation, problem-solving, and user-focused design — skills relevant to the medical field.

How It Works

  • User submits medication details via Google Form.
  • Responses are saved in Google Sheets.
  • Google Apps Script reads the data and creates calendar events for each reminder.
  • Each reminder includes an invitation to the user’s Google Calendar.
  • One confirmation email is sent listing all reminder dates and times.
