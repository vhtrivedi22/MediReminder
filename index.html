MediReminder

Automated Medication Reminder System

Project Description

MediReminder is a web-based tool that helps users remember their medications efficiently and easily. Through the chaos of daily routines, busy jobs, and working schedules, remembering medication can sometimes be difficult. Using Google Forms, Sheets, Calendar, and Gmail, MediReminder automatically schedules reminders and sends confirmation emails to ensure that you are consistent and healthy.

How It Works

  • User submits medication details via Google Form.
  • Responses are saved in Google Sheets.
  • Google Apps Script reads the data and creates calendar events for each reminder.
  • Each reminder includes an invitation to the user’s Google Calendar.
  • One confirmation email is sent listing all reminder dates and times.
